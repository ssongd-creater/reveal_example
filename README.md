웹 실습 사이트 제작
Reveal Site는 기존 제작 되어 있는 사이트를 재구성한 사이트입니다.

JQuery를 이용하여 Scroll Event / Click Event 를 구현하였습니다.

Library는 venobox를 사용하여 카테고리 분류와 자동 slide 기능은 owlcarousel 를 사용하였습니다.

## 사용한 언어

- [HTML]
- [CSS]
- [jQuery]
- [PHP]
- [XAMPP]
